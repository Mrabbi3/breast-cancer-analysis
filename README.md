## Project overview

This project performs a statistical analysis of the Wisconsin Diagnostic Breast Cancer (WDBC) dataset to identify which tumor features are most useful for distinguishing malignant from benign cases. We analyze 569 tumor samples described by 30 numeric features each, looking at distributions, group differences, correlations, hypothesis tests, and feature importance.

## Dataset

**Source:** [Breast Cancer Wisconsin (Diagnostic) Data Set](https://www.kaggle.com/datasets/uciml/breast-cancer-wisconsin-data) on Kaggle, originally from the UCI Machine Learning Repository (W. Wolberg, University of Wisconsin, 1995).

**Setup:** Download `data.csv` from the Kaggle page, rename it to `wdbc.csv`, and place it in `data/wdbc.csv`. The cleaned version (`wdbc_clean.csv`) is generated automatically by the EDA notebook.

## Group contributions

- **Member A — Data & Distributions:** Data loading, cleaning, quality checks, descriptive statistics by group, distribution and boxplot visualizations. (`notebooks/01_eda_memberA.ipynb`)
- **Member B — Correlations & Hypothesis Testing:** Correlation matrix, pairplots, t-tests, Mann-Whitney U tests, effect sizes, normality checks. (`notebooks/02_correlation_hypothesis_memberB.ipynb`)
- **Member C — Feature Importance & Reporting:** Feature ranking by AUC / mutual information, logistic regression importances, PCA, final report compilation. (`notebooks/03_feature_importance_memberC.ipynb`)

## How to reproduce

1. Clone the repo
2. Install dependencies: `pip install pandas numpy matplotlib seaborn scipy scikit-learn jupyter`
3. Download `data.csv` from the Kaggle link above, rename to `wdbc.csv`, place in `data/`
4. Open `notebooks/01_eda_memberA.ipynb` in Jupyter and run all cells
5. Then run notebooks 02 and 03 in order

## Key findings (preview)

- The dataset has 569 samples (357 benign, 212 malignant) with no missing values
- The `worst` family of features dominates the discrimination between malignant and benign tumors
- The single most discriminative feature is `concave_points_worst` (standardized mean difference ≈ 2.6)
- Size features (radius, perimeter, area) are highly inter-correlated, indicating redundancy
